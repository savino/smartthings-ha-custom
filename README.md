# SmartThings Custom
A fork of the Home Assistant SmartThings Integration. This adds better support for Samsung AC Devices.

## Fixed features:
  - AC Preset modes (shows all the presets available)

## Installation:
### HACS
- Remove your original smartthings integration if you have one set up (optional)
- Add `https://github.com/savino/smartthings-ha-custom` as a Custom Repository
- Install `SmartThings Custom` from the HACS Integrations tab
- Restart Home Assistant
- Install `SmartThings` from the HA Integrations tab
- Enjoy!

### Manually
- Remove your original smartthings integration if you have one set up (optional)
- Copy the smartthings folder to custom_components\
- Restart Home Assistant
- Install `SmartThings` from the HA Integrations tab
- Enjoy!
